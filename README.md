# la-biblioteca-de-alejandria

Public index for Observacionismo for Local-First AI Agents.

Status: `PUBLIC_GITHUB_LOW_CLAIM_INDEX`.

This is the door for people who do not know the MEDIOEVO ecosystem yet. It
keeps the map small: one core, one execution wrapper, one handoff protocol, one
dashboard/API direction, and one research lab.

## Start Here

Observacionismo for AI agents is an engineering method:

- observe before acting;
- attach evidence before claims;
- route risky actions through explicit receptors;
- write witness logs for replay and handoff;
- keep falsifiers next to research hypotheses.

It does not claim to prove new physics, consciousness, social prediction,
medical safety, or autonomous agent safety.
