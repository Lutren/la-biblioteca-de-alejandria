# Whitepaper Draft

## Summary

Observacionismo for Local-First AI Agents is a public-safe engineering method:
evidence before action, explicit receptors for risky tool use, witness logs for
handoff, and falsifiers for research claims.

The shortest public version:

> `obsai-core` is the core. `safe-exec` wraps tools. `agent-handoff-protocol`
> keeps sessions continuous. `residueos` visualizes the review layer.
> `duat-lab` tests speculative models without presenting them as truth.

## Why Newton Matters

Newton is not treated here as an enemy. Newton is treated as a useful local
compression: powerful inside the regime where its assumptions hold, misleading
when the regime label disappears.

The AI-agent translation is direct. A model output can be useful and still be
bounded. A decision can be efficient and still need a witness log. A hypothesis
can be creative and still require falsifiers before publication.

## Practical Claim

The practical claim is not "new physics." The practical claim is:

- high-risk actions should carry evidence and authorization;
- public claims should carry sources and uncertainty;
- handoffs should carry state, decisions and open residue;
- research models should carry tests that can fail them.
