# Roadmap

## Now

- Keep `obsai-core` as the canonical core.
- Keep public claims low and evidence-backed.
- Add executable demos that show `APPROVE`, `REVIEW` and `BLOCK`.
- Add witness-log examples that can be replayed without private data.

## Next

- Build a 100-500 scenario benchmark for agent actions.
- Add examples for LangChain, AutoGen and CrewAI.
- Add a short video demo: destructive action blocked, read-only action allowed,
  witness log produced.
- Add a public replay report template.

## Later

- Calibrate thresholds with larger scenario sets.
- Add dashboard/API surfaces through `residueos`.
- Add synthetic falsifier suites through `duat-lab`.
- Publish short technical essays only when each claim has a boundary and a
  falsifier.

## Not On This Roadmap

- Publishing private runtime internals.
- Publishing RPG, TCG, book drafts, prompts, sessions or secrets.
- Publishing validated physics-replacement, consciousness or medical-safety
  claims.
